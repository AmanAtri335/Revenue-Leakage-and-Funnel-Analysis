## Data Source

This project uses the **Olist Brazilian E-Commerce Dataset**, a real-world dataset available on Kaggle. It contains transactional data covering the complete order lifecycle, including customers, orders, payments, products, and delivery information.

Dataset Link:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

## How to Download the Dataset

###Direct Download (Recommended)

1. Open the dataset link above
2. Sign in to Kaggle (required)
3. Click on the **“Download”** button
4. Extract the downloaded ZIP file
5. Load the CSV files into your SQL environment or Power BI

## Note

The dataset consists of multiple relational tables and requires joining using keys such as:

* `order_id`
* `customer_id`
* `product_id`

Data cleaning and transformation were performed using SQL before visualization in Power BI.
